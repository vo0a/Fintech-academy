# DaromToken

    balances, totalSupply : 만들고자 하는 토큰의 전체갯수 ('0' 18개를 제외한 숫자, 1000개)
    name : 만들고자 하는 토큰명 (DaromCoin)
    decimals : 이더 단위와 일치하기 때문에 디폴트 18로 설정 
    symbol : 토큰 단위
    unitsOneEthCanBuy : 1이더를 전송하면 10토큰을 받음(해당 컨트랙트 주소로 이더를 송금하였을경우)
